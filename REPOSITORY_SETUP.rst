================
Repository Setup
================

Notes for creating this repository.

Initialize package with cookiecutter
------------------------------------

Followed the official instruction of cookiecutter.
https://cookiecutter.readthedocs.io/en/latest/README.html

::

 $ cookiecutter https://github.com/audreyfeldroy/cookiecutter-pypackage

GitHub settings
---------------

Change default branch to ``develop``.
https://github.com/amano-takahisa/gravel/settings/branches

This change default target branch when PR is merged in GitHub.
