============
Installation
============

Stable release
--------------

Stable release is not avaliable

.. Stable release is not avaliable

    To install gravel, run this command in your terminal:

    .. code-block:: console

        $ pip install gravel

    This is the preferred method to install gravel, as it will always install the most recent stable release.

    If you don't have `pip`_ installed, this `Python installation guide`_ can guide
    you through the process.

    .. _pip: https://pip.pypa.io
    .. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for gravel can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/amano-takahisa/gravel

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/amano-takahisa/gravel/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ pip install -e .


.. _Github repo: https://github.com/amano-takahisa/gravel
.. _tarball: https://github.com/amano-takahisa/gravel/tarball/master
